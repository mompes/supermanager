Librería del Supermanager ACB
=============================
¿Qué es?
--------
Es una librería Java para Android que permite interactuar con el supermanager de la ACB (http://supermanager.acb.com)

Permite llevar a acabo acciones como: loguearse, descargar los datos de los equipos, descargar datos de jugadores, vender y comprar jugadores,...

Es la librería utilizada por la aplicación SMDroid (https://play.google.com/store/apps/details?id=es.mompes.supermanager)
